# School Election Voting System

A complete digital voting platform developed for school elections.

## Features

- Student Verification
- Teacher Voting
- Management Voting
- Firebase Hosting
- Firestore Integration
- Live Results Dashboard
- Secure One Vote System

## Technologies Used

- HTML
- CSS
- JavaScript
- Firebase Hosting
- Firestore Database
